search(query: str, k: int = None) → List[Dict[str, str]]¶

Search returns a list of matching book summaries in order of relevance.

Usage : Instantiate Scout class, and simply search. ` q = Scout("dbname") q.search("The jumping $$ fox changes color.") ` The above snippet returns upto 5 results. To control the number of results, ‘k’ int. ` q.search("The jumping $$ fox changes color.", k=10) `

Parameters

  • query (str) – A search query to find documents.

  • k (int, optional) – Maximum number of results, defaults to self.max_results

Raises

  • UnsupportedQueryType – Raised when the search query is not str.

  • EmptyQuery – Raised when the search query is empty.

  • LookupError – Raised when no results are found.

Returns

List of book {id, summary} in order of relevance.

Return type

List[Dict[str, str]]

scout.term¶

scout.term.tokenize¶

term.tokenize() → List[str]¶

Tokenization is a process of text simplification.

This function returns a list of important words in a text. The input text is converted to all lowercase characters. Special characters and punctuations are removed from the text. Finally, stop words like ‘a’, ‘the’, ‘for’, ‘from’ etc. are removed.

> Note : Hyphenated words are converted to single word. > For example, ‘un-filtered’ would become ‘unfiltered’.

Parameters

text (str) – Input text to be tokenized.

Raises

TypeError – Input text variable must be of type ‘str’.

Returns

List of tokenized words.

Return type

List[str]
